::: section
::: section-title
Definition of a Group
:::

A **group** (G, ┬╖) is a set G with a binary operation ┬╖ satisfying: (1) **Associativity:** (a┬╖b)┬╖c = a┬╖(b┬╖c) for all a, b, c тИИ G. (2) **Identity:** There exists e тИИ G with e┬╖a = a┬╖e = a for all a. (3) **Inverses:** For each a тИИ G, there exists aтБ╗┬╣ with a┬╖aтБ╗┬╣ = aтБ╗┬╣┬╖a = e. The group is **abelian** (commutative) if additionally a┬╖b = b┬╖a for all a, b.

The **order** of G, written \|G\|, is the number of elements. The **order** of an element g, written ord(g), is the smallest positive integer n with gтБ┐ = e (or тИЮ if no such n exists).
:::

::: section
::: section-title
Subgroups
:::

A **subgroup** H тЙд G is a nonempty subset closed under the group operation and inverses. **Subgroup test:** H тЙд G if and only if H тЙа тИЕ and abтБ╗┬╣ тИИ H for all a, b тИИ H. For finite H: just check closure (inverses are automatic).

The subgroup **generated by** a set S, written тЯиSтЯй, is the smallest subgroup containing S. A **cyclic group** G = тЯиgтЯй is generated by a single element. Every cyclic group is abelian. Up to isomorphism, the cyclic groups are тДд (infinite) and тДд/nтДд (finite, order n).
:::

::: section
::: section-title
Cayley\'s Theorem
:::

**Cayley\'s Theorem:** Every group G is isomorphic to a subgroup of Sym(G), the group of all permutations of the set G. In particular, every finite group of order n embeds into S_n. This is proved by the left-multiplication action: each g тИИ G defines a permutation x тЖж gx.
:::

Cosets and Lagrange\'s Theorem
:::

::: section
::: section-title
Cosets
:::

For H тЙд G and g тИИ G, the **left coset** gH = {gh : h тИИ H} and **right coset** Hg = {hg : h тИИ H}. Key facts: (1) gH = kH if and only if gтБ╗┬╣k тИИ H. (2) Distinct left cosets are disjoint. (3) All left cosets have the same size \|H\|. (4) The left cosets partition G. The number of distinct left cosets is the **index** \[G:H\] = \|G\|/\|H\|.

::: {.callout .warn}
::: callout-label
Left тЙа Right
:::

In general, gH тЙа Hg. Left and right cosets coincide if and only if H is a normal subgroup.
:::
:::

::: section
::: section-title
Lagrange\'s Theorem
:::

**Lagrange\'s Theorem:** If H тЙд G and G is finite, then \|H\| divides \|G\|. Moreover, \|G\| = \[G:H\] ┬╖ \|H\|.

**Corollaries:** (1) The order of every element divides \|G\|. (2) g^\|G\|^ = e for all g тИИ G. (3) Every group of prime order is cyclic (and simple). (4) Fermat\'s little theorem: a^p^ тЙб a (mod p) for prime p. (5) Euler\'s theorem: a^╧Ж(n)^ тЙб 1 (mod n) for gcd(a,n) = 1.

::: {.callout .key}
::: callout-label
The converse is false
:::

Lagrange says \|H\| divides \|G\|, but not every divisor of \|G\| is the order of some subgroup. The smallest counterexample: AтВД has order 12 but no subgroup of order 6. Cauchy\'s and Sylow\'s theorems provide partial converses.
:::
:::

::: section
::: section-title
Cauchy\'s Theorem
:::

**Cauchy\'s Theorem:** If p is prime and p divides \|G\|, then G contains an element of order p (and hence a cyclic subgroup of order p). This is the weakest partial converse to Lagrange; Sylow generalizes it.
:::

::: section
::: section-title
Normal Subgroups
:::

A subgroup N тЙд G is **normal**, written N тЧБ G, if gNgтБ╗┬╣ = N for all g тИИ G. Equivalently: gN = Ng for all g, i.e., left and right cosets coincide. Every subgroup of an abelian group is normal. Every subgroup of index 2 is normal. The kernel of any homomorphism is normal.
:::

::: section
::: section-title
Quotient Groups
:::

If N тЧБ G, the set of cosets G/N = {gN : g тИИ G} forms a group under the operation (gN)(hN) = (gh)N. This is the **quotient group** (or factor group). Its order is \|G/N\| = \[G:N\] = \|G\|/\|N\|. The quotient construction is only well-defined when N is normal --- this is why normality matters.

The natural projection ╧А: G тЖТ G/N defined by ╧А(g) = gN is a surjective homomorphism with kernel N.
:::

Homomorphisms and Isomorphism Theorems
:::

::: section
::: section-title
Homomorphisms
:::

A **group homomorphism** f: G тЖТ H satisfies f(ab) = f(a)f(b) for all a, b тИИ G. It follows that f(e_G) = e_H and f(aтБ╗┬╣) = f(a)тБ╗┬╣. The **kernel** ker(f) = {g тИИ G : f(g) = e_H} is a normal subgroup of G. The **image** im(f) = f(G) is a subgroup of H. f is injective iff ker(f) = {e}. An **isomorphism** is a bijective homomorphism; an **automorphism** is an isomorphism from G to itself.
:::

::: section
::: section-title
The Isomorphism Theorems
:::

::: {.callout .key}
::: callout-label
The three isomorphism theorems
:::

**First (Fundamental):** If f: G тЖТ H is a homomorphism, then G/ker(f) тЙЕ im(f). This is the most important theorem in the subject --- it says that every homomorphic image of G is isomorphic to a quotient of G.\
\
**Second:** If H тЙд G and N тЧБ G, then HN тЙд G, HтИйN тЧБ H, and HN/N тЙЕ H/(HтИйN).\
\
**Third:** If N тЙд K тЙд G with N тЧБ G and K тЧБ G, then (G/N)/(K/N) тЙЕ G/K.\
\
**Correspondence (Fourth):** If N тЧБ G, there is an inclusion-preserving bijection between subgroups of G containing N and subgroups of G/N, given by H тЖж H/N.
:::

Direct and Semidirect Products
:::

The **direct product** G ├Ч H has elements (g, h) with componentwise operation: (gтВБ, hтВБ)┬╖(gтВВ, hтВВ) = (gтВБgтВВ, hтВБhтВВ). If G = NтВБ ├Ч ┬╖┬╖┬╖ ├Ч N_k internally (N_i тЧБ G, N_i тИй (product of others) = {e}, G = NтВБ┬╖┬╖┬╖N_k), then G тЙЕ NтВБ ├Ч ┬╖┬╖┬╖ ├Ч N_k.

The **semidirect product** N тЛК\_╧Ж H is defined when N тЧБ G, H тЙд G, N тИй H = {e}, and G = NH. The action of H on N by conjugation is encoded by a homomorphism ╧Ж: H тЖТ Aut(N). Every group that is a product of a normal subgroup and a complement is a semidirect product. When ╧Ж is trivial, the semidirect product reduces to the direct product.

Group Actions
:::

::: section
::: section-title
Definitions
:::

A **group action** of G on a set X is a map G ├Ч X тЖТ X, written (g, x) тЖж g┬╖x, satisfying: (1) e┬╖x = x for all x, and (2) g┬╖(h┬╖x) = (gh)┬╖x for all g, h, x. The **orbit** of x is Orb(x) = {g┬╖x : g тИИ G}. The **stabilizer** of x is Stab(x) = {g тИИ G : g┬╖x = x}, which is a subgroup of G.
:::

::: section
::: section-title
Orbit-Stabilizer Theorem
:::

**Theorem:** \|G\| = \|Orb(x)\| ┬╖ \|Stab(x)\| for any x тИИ X. Equivalently, the number of elements in the orbit of x equals the index \[G : Stab(x)\].

**Burnside\'s Counting Lemma** (Cauchy-Frobenius): The number of distinct orbits under a group action is (1/\|G\|) тИС~gтИИG~ \|Fix(g)\|, where Fix(g) is the set of elements fixed by g. Used for counting distinct objects under symmetry (e.g., distinct necklaces, colorings).
:::

::: section
::: section-title
Important Actions
:::

**Conjugation action:** G acts on itself by g┬╖x = gxgтБ╗┬╣. Orbits are **conjugacy classes**. Stabilizer of x is the **centralizer** C_G(x). The **class equation**: \|G\| = \|Z(G)\| + тИС \[G : C_G(x_i)\], summing over representatives of non-central conjugacy classes. Here Z(G) is the **center** (elements commuting with everything).

**Action on cosets:** G acts on G/H by left multiplication. This action yields Cayley\'s theorem (H = {e}) and is the key tool in Sylow theory.
:::

::: {#ch-sylow .chapter}

::: chapter-title
Sylow Theorems
:::

Let G be a finite group with \|G\| = p^n^m where p is prime and p does not divide m. A **Sylow p-subgroup** is a subgroup of order p^n^ --- the maximum power of p dividing \|G\|.

::: {.callout .key}
::: callout-label
The three Sylow theorems
:::

**Sylow I (Existence):** G has at least one Sylow p-subgroup. Moreover, every p-subgroup of G is contained in some Sylow p-subgroup.\
\
**Sylow II (Conjugacy):** All Sylow p-subgroups of G are conjugate to each other. In particular, they are all isomorphic.\
\
**Sylow III (Counting):** The number n_p of Sylow p-subgroups satisfies: (a) n_p тЙб 1 (mod p), and (b) n_p divides m = \|G\|/p^n^. Also, n_p = \[G : N_G(P)\] where N_G(P) is the normalizer of any Sylow p-subgroup P.
:::

::: {.callout .tip}
::: callout-label
Using Sylow to analyze groups
:::

The standard technique: (1) Factor \|G\| = pтВБ^aтВБ^ ┬╖ pтВВ^aтВВ^ ┬╖ ┬╖┬╖┬╖ (2) For each prime p_i, list possible values of n\_{p_i} using Sylow III. (3) If n_p = 1, the Sylow p-subgroup is normal (unique implies normal, since conjugation would produce another). (4) Use normality and product structure to decompose G. This technique classifies all groups of many small orders.
:::

**Application:** Every group of order 15 is cyclic. Proof: nтВГ divides 5 and nтВГ тЙб 1 (mod 3), so nтВГ = 1. nтВЕ divides 3 and nтВЕ тЙб 1 (mod 5), so nтВЕ = 1. Both Sylow subgroups are normal, their intersection is trivial (coprime orders), so G тЙЕ тДд/3тДд ├Ч тДд/5тДд тЙЕ тДд/15тДд.
:::

**Fundamental Theorem of Finite Abelian Groups:** Every finite abelian group is isomorphic to a direct product of cyclic groups of prime power order: G тЙЕ тДд/pтВБ^aтВБ^тДд ├Ч тДд/pтВВ^aтВВ^тДд ├Ч ┬╖┬╖┬╖ ├Ч тДд/p_k^a_k^тДд. ::: section
::: section-title
Commutator Subgroup and Solvability
:::

The **commutator** of a, b is \[a,b\] = abaтБ╗┬╣bтБ╗┬╣. The **commutator subgroup** (derived subgroup) G\' = \[G,G\] = тЯи{\[a,b\] : a,b тИИ G}тЯй is the smallest normal subgroup N such that G/N is abelian. The **derived series** G тЙе G\' тЙе G\'\' тЙе ┬╖┬╖┬╖ iterates this. G is **solvable** if the derived series reaches {e} in finitely many steps. All abelian groups are solvable. S_n is solvable for n тЙд 4 but not for n тЙе 5 (because AтВЕ is simple and non-abelian).
:::

::: section
::: section-title
Simple Groups
:::

A group G is **simple** if its only normal subgroups are {e} and G. Simple groups are the \"atoms\" of group theory --- every group can be built from simple groups via extensions (Jordan-H├╢lder theorem: the composition factors are unique up to order). The **classification of finite simple groups** (completed \~2004, tens of thousands of pages across hundreds of papers): every finite simple group is one of: (1) тДд/pтДд for prime p, (2) the alternating group A_n for n тЙе 5, (3) a group of Lie type, or (4) one of 26 sporadic groups (including the Monster, of order тЙИ 8 ├Ч 10тБ╡┬│).
:::

::: chapter-title
Bridge to Rings and Fields
:::

Groups are the first rung of the algebraic hierarchy. Each successive structure adds more axioms:

  Structure              Operations             Key Addition Beyond Group
  ---------------------- ---------------------- -------------------------------------------------------------------------------------------
  **Group**              One (┬╖)                ---
  **Ring**               Two (+, ┬╖)             (R, +) is abelian group; ┬╖ is associative and distributes over +
  **Commutative ring**   Two                    Multiplication is commutative
  **Integral domain**    Two                    No zero divisors (ab = 0 тЗТ a = 0 or b = 0)
  **Field**              Two                    (F \\ {0}, ┬╖) is also an abelian group (every nonzero element has multiplicative inverse)
  **Vector space**       Two (+ and scalar ┬╖)   Group (V, +) with scalar multiplication from a field F

The group-theoretic concepts of subgroup, quotient, homomorphism, and isomorphism theorem all have exact analogs in ring theory (subrings, ideals, quotient rings, ring homomorphisms) and module theory. The language learned in group theory is the language of all algebra.

::: {.callout .note}
::: callout-label
Why this hierarchy matters
:::

Many problems that appear to be about rings or fields are solved by extracting a group-theory problem from within them. The multiplicative group of a field, the group of units of a ring, the automorphism group of a field extension --- these are all group-theory objects living inside richer structures. Galois theory, the crown jewel of algebra, is precisely the study of field extensions through their automorphism groups.
:::

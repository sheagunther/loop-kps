# Knowledge Pack Index

**Purpose:** This index lets AI instances identify which knowledge packs to request from the Operator based on task requirements. Packs live in /Files as markdown and/or L21 HTML documents. The Operator swaps them in and out as needed. Scan this index first; request specific packs by name.

**Protocol:** If a task requires specialized knowledge not in your current context, check this index. If a relevant pack exists, tell the Operator: "I could use the [Pack Name] knowledge pack for this." The Operator will load it. Do not guess at content you haven't read.

**Current count:** 70 packs across 3 active layers. Last updated: 1 May 2026.

**Organization:** Packs are organized by Architecture layer and Core branch, per KP Architecture v1. Layers 0–2 ship invisibly with every product. Layer 3 is the visible product catalog.
